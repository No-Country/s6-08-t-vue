# DonaFunding
Este proyecto fue diseñado para personas que están pasando una situación especial y no la pueda cubrir economicamente.

Podrás donar y ayudar a la causa que más se identifique con vos.


### Design

-  *Figma*
- *Canva*


### Frontend

- *Vue*
- *Pinia*
- *Tailwind*
- *Css*

### Backend

- *NestJs*
- *Passport*
- *MercadoPago*
- *Typeorm*
- *Postgres SQL*
